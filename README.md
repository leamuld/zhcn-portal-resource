# zhcn-portal-resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面不针对任何特定域名或网站，仅作为静态资源存放与展示使用。适合需要集中管理、版本控制或简单部署 HTML 文件的场景。

## 目录说明

```
zhcn-portal-resource/
├── pages/          # 存放具体 HTML 页面文件
├── assets/         # 页面引用的静态资源（如图片、样式、脚本）
└── README.md       # 本文件
```

- **pages/**：每个 HTML 文件为一个独立页面，文件名通常反映其内容或用途。
- **assets/**：按需存放页面所需的公共或私有资源，可进一步按类型或页面组织子目录。

## 页面归档说明

- 所有 HTML 页面均为独立文件，可直接在浏览器中打开运行。
- 页面之间无依赖关系，可单独更新或替换。
- 归档内容可能不定期更新，但不会针对特定网站或服务进行定制。
- 如需新增页面，请将文件放入 `pages/` 目录，并在必要时更新相关资源路径。

## 维护说明

- 本仓库由维护者不定期更新，不承诺固定发布周期。
- 欢迎通过 Issues 提交建议或报告问题。
- Pull Request 需确保页面独立可用，且不引入外部不可控资源。
- 本仓库仅供归档与展示用途，不提供任何形式的服务保证。

## 使用方式

1. 克隆或下载本仓库。
2. 直接打开 `pages/` 下的 HTML 文件，或在任意静态服务器中部署。
3. 如需引用资源，请使用相对路径，确保页面可独立运行。
